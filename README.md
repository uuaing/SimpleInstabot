# SimpleInstabot
A simple bot for automatically like, follow, comment, unfollow on Instagram

### How to start
1. Install Python 2.7 or 3.6 installed
2. Install requests: 
```bash 
pip install requests
```
3. Add username/password to example.py
4. run bot: 
```bash 
python example.py 
```
or 
```bash
python3 example.py
```
